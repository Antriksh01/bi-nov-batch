HTML - Day 2

HTML Block and Inline Tags/Elements

Block Level Tags/Elements

  1. Block level tags always starts with new line
  2. Block level tags takes full width of a web page
  3. Block level tags has a top and bottom margin

List of block level tags:

  • div
  • h1-h6
  • header
  • footer
  • nav
  • main
  • table
  • ul
  • ol
  • form
  • article
  • section

Inline Tags/Elements

  • Inline tags never starts with new line
  • Inline tags takes as much as required space of a web page

List of Inline Tags/Elements

  • a
  • span
  • input
  • label
  • img
